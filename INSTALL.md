
Installation
============

"AppRealKit" is distributed both as binaries and as source codes (example).
It is simple to use by integrating AppRealKit into yout iOS projects.

Integrating from the Source Code
-----------------------------

Integrating Etool requires:
* Xcode
* iOS SDK
